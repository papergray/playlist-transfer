# 🎵 Playlist Transfer — Android APK

Transfer playlists between Spotify, Apple Music, YouTube Music, and Tidal — including between accounts on the same service.

---

## Prerequisites

Install these once on your machine:

| Tool | Download |
|------|----------|
| **Node.js** (v18+) | https://nodejs.org |
| **Git** | https://git-scm.com |
| **Android Studio** | https://developer.android.com/studio |
| **Java JDK 17** | https://adoptium.net |

---

## Step 1 — Push to GitHub

```bash
# 1. Create a new repo on github.com (name it "playlist-transfer"), then:

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/playlist-transfer.git
git push -u origin main
```

---

## Step 2 — Install dependencies

```bash
npm install
```

---

## Step 3 — Build the web app

```bash
npm run build
```

This creates a `dist/` folder with the compiled app.

---

## Step 4 — Add Android platform

```bash
npx cap add android
```

This generates the `android/` folder — a full Android Studio project.

---

## Step 5 — Sync web build into Android

```bash
npx cap sync android
```

Run this every time you change the web code.

---

## Step 6 — Open in Android Studio

```bash
npx cap open android
```

Android Studio will open the project automatically.

---

## Step 7 — Build the APK

Inside Android Studio:

1. Wait for Gradle to sync (bottom progress bar)
2. Go to **Build → Build Bundle(s) / APK(s) → Build APK(s)**
3. Click **"locate"** in the notification that pops up
4. Your APK is at:
   ```
   android/app/build/outputs/apk/debug/app-debug.apk
   ```

---

## Step 8 — Install on your phone

**Option A — USB:**
```bash
# Enable USB Debugging on your phone first, then:
adb install android/app/build/outputs/apk/debug/app-debug.apk
```

**Option B — File transfer:**
- Copy `app-debug.apk` to your phone
- Open it from your file manager
- Allow "Install from unknown sources" if prompted

---

## Workflow for future updates

```bash
# After editing src/App.jsx:
npm run build
npx cap sync android
# Then rebuild APK in Android Studio
```

---

## Troubleshooting

**Gradle sync fails:**
- Make sure JDK 17 is set in Android Studio → Settings → Build Tools → Gradle → Gradle JDK

**`adb` not found:**
- Add Android SDK platform-tools to your PATH:
  - Mac/Linux: `export PATH=$PATH:~/Library/Android/sdk/platform-tools`
  - Windows: Add `C:\Users\YOU\AppData\Local\Android\Sdk\platform-tools` to System PATH

**App crashes on open:**
- Run `npx cap sync android` again before rebuilding

## Project Structure

```
playlist-transfer/
├── src/
│   ├── main.jsx          # React entry point
│   └── App.jsx           # Main app (all UI + logic)
├── index.html            # HTML shell
├── vite.config.js        # Vite bundler config
├── capacitor.config.json # Android app config (app ID, name)
├── package.json          # Dependencies
└── android/              # Generated by Capacitor (don't edit manually)
```

---

## Change App ID / Name

Edit `capacitor.config.json`:
```json
{
  "appId": "com.yourname.playlisttransfer",
  "appName": "My Playlist App"
}
```

Then re-run `npx cap sync android`.
